# VEFS Programs Page Requirements

**Status:** In Progress
**Priority:** High
**Created:** 2025-12-23

## Page Overview

The Programs Page showcases VEFS's core initiatives and offerings. It serves as the central hub for understanding what the foundation does and how different community segments can benefit from its programs. The page should inspire action and provide easy navigation to deeper program information.

## Page Structure & Sections

### 1. Page Header
- **Status:** Required
- **Content:**
  - Page title: "Our Programs" or "What We Do"
  - Hero image or background (VEFS programs in action)
  - Subheading: Brief tagline (e.g., "Empowering Communities Through Ecological Education")
- **Design Notes:**
  - Consistent header styling with other pages
  - Full-width banner with background image
  - Text overlay for readability

---

### 2. Introduction Section
- **Status:** Required
- **Format:** Text block with optional visual element
- **Content:**
  - Brief introduction (2-3 sentences) explaining:
    - VEFS's approach to programs
    - How programs address ecological and social goals
    - Why community engagement matters
    - Connection to indigenous knowledge and Tamil heritage
- **Design Notes:**
  - Engaging, accessible language
  - Optional: Supporting icon or illustration
  - Optional: Pull-quote from founder or team member

---

### 3. Programs by Target Audience
- **Status:** Required
- **Format:** 4 main audience sections with expandable program cards
- **Content:**

Programs organized into 4 audience categories:

#### A. Programs for Students
**Purpose:** Educational programs designed for school and college students

Structure for each program:
- Program title
- Brief description (1-2 sentences)
- Program image or icon
- Key highlights (2-3 bullet points)
- "Learn More" button → Opens detailed modal/page
- Optional: Age range or grade level

**Typical Student Programs:**
- Environmental awareness workshops
- School partnership programs
- Outdoor nature-based learning experiences
- Nature conservation competitions
- Tree planting campaigns

#### B. Programs for Farmers
**Purpose:** Support agricultural communities in adopting sustainable practices

Structure for each program:
- Program title
- Brief description (1-2 sentences)
- Program image or icon
- Key benefits (2-3 bullet points)
- "Learn More" button
- Optional: Season/timing information

**Typical Farmer Programs:**
- Indigenous tree cultivation training
- Sustainable farming practices
- Forest gardening systems
- Cooperative partnership initiatives
- Income generation from ecological farming

#### C. Programs for Women
**Purpose:** Empower women through environmental education and economic opportunity

Structure for each program:
- Program title
- Brief description (1-2 sentences)
- Program image or icon
- Key focuses (2-3 bullet points)
- "Learn More" button
- Optional: Time commitment

**Typical Women's Programs:**
- Women's environmental leadership
- Sustainable livelihood programs
- Women in agriculture initiatives
- Health and wellness through nature
- Women's self-help groups in ecology

#### D. Programs for General Public
**Purpose:** Community-wide engagement and environmental awareness

Structure for each program:
- Program title
- Brief description (1-2 sentences)
- Program image or icon
- Who can join (eligibility or open to all)
- "Learn More" button
- Optional: Next session date

**Typical Public Programs:**
- Community tree planting events
- Eco-tourism and nature trails
- Open awareness seminars
- Environmental forums
- Public participation in conservation projects

### 4. Program Card Design

Each program is displayed as a card with:

**Visual Elements:**
- Program image (400x300px or 4:3 aspect ratio)
- Program title (H4 heading, sage green)
- Brief description text (small, 1-2 sentences)
- Key highlights/bullets (2-3 items)
- Program icon (optional, sage green)

**Interactive Elements:**
- "Learn More" button (secondary style)
- Hover effect: Card lift, shadow increase
- Click action: Opens detailed program information

**Design Specification:**
- Card padding: 24px (lg)
- Border radius: 8px
- Background: White with 1px subtle border
- Grid layout: 2 columns (desktop), 1 column (mobile)
- Gap between cards: 24px (lg)

---

### 5. Program Detail Modal/Page
- **Status:** Required
- **Triggered by:** "Learn More" button on program card
- **Format:** Modal overlay or dedicated page
- **Content for each program:**
  - Program title (H2)
  - Full program description (2-4 paragraphs)
  - **Objectives:** What the program aims to achieve
  - **Activities:** Specific activities/workshops included
  - **Target Audience:** Who benefits from this program
  - **Duration:** Program length (ongoing, seasonal, etc.)
  - **Schedule:** When programs are offered
  - **Location:** Where activities take place
  - **Cost:** Free, fee-based, or sliding scale (if applicable)
  - **Facilitators/Trainers:** Names and brief bios
  - **Outcomes:** Expected results or learning outcomes
  - **Impact Metrics:** How many beneficiaries, results achieved
  - **Program Images/Gallery:** Photos from past programs
  - **Testimonials:** Quotes from participants (optional)
  - **Next Steps:** Call-to-action (Register, Contact, etc.)

**Modal Design Notes:**
- Close button (X) in top-right
- Responsive: Adapt for mobile viewing
- Optional: Breadcrumb navigation (Home > Programs > [Program Name])
- Background: Sage green accent color or gradient
- Content: Scrollable if needed

## Interactions & Features

### Program Card Interactions
- **Hover:** Card elevation, shadow increase, slight scale transform
- **Click:** Opens detailed modal with full program information
- **Focus:** Clear focus outline for keyboard navigation

### Modal Interactions
- **Open:** Smooth fade-in, slight scale animation
- **Close:** Fade-out on close button click or outside click
- **Navigation:** Optional: Previous/Next program buttons within modal
- **Scroll:** Allow scrolling within modal if content is long

### Responsive Behavior
- **Desktop:** 2-column grid, full details visible on hover
- **Tablet:** 1-2 column grid, touch-friendly cards
- **Mobile:** 1-column stack, larger touch targets

## Design & Visual Approach

### Color Usage
- **Headers:** Sage green (#6B8E23)
- **Card backgrounds:** White with subtle border
- **Audience section dividers:** Light gray (#F5F5F5)
- **"Learn More" buttons:** Secondary style (sage green border, white background)
- **"Get Involved" CTA:** Golden/amber buttons
- **Icons:** Sage green

### Layout
- **Max content width:** 1200px
- **Card grid gap:** 24px (lg)
- **Section padding:** 64px vertical on desktop, 32px on mobile
- **Card aspect ratio:** 4:3 for images

### Typography
- **Section headings (H2):** Serif, 36px, sage green
- **Audience subheadings:** Serif, 28px, sage green
- **Card titles (H4):** Sans-serif, 20px, sage green
- **Card descriptions:** Sans-serif, 16px, dark gray
- **Button text:** Sans-serif, 16px, semibold

---

## Responsive Design

### Desktop (1024px+)
- 2-column card grid
- Full details visible in modals
- Maximum content width: 1200px

### Tablet (768px - 1023px)
- 1-2 column grid (adapting to content)
- Touch-friendly card sizing
- Adjusted padding

### Mobile (< 768px)
- 1-column stack
- Full-width cards
- Larger touch targets (minimum 44x44px)
- Hamburger navigation
