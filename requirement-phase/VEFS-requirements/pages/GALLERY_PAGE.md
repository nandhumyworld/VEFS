# VEFS Gallery Page Requirements

**Status:** In Progress
**Priority:** High
**Created:** 2025-12-23

## Page Overview

The Gallery Page showcases VEFS's visual history and impact through a curated collection of photographs from events, programs, trainings, and field activities. It serves as both a visual portfolio and an inspirational resource, demonstrating the foundation's work across communities and ecosystems.

## Page Structure & Sections

### 1. Page Header
- **Status:** Required
- **Content:**
  - Page title: "Gallery" or "Photo Gallery" or "Our Journey in Pictures"
  - Hero image or background (inspiring nature/community photo)
  - Subheading: Brief tagline (e.g., "Witness our Impact Through Images")
- **Design Notes:**
  - Consistent header styling
  - Full-width hero image (striking visual)
  - Text overlay for readability

---

### 2. Introduction Section
- **Status:** Required
- **Format:** Brief text block
- **Content:**
  - Short introduction (1-2 sentences) about the gallery
  - Explanation of what visitors will see
  - Optional: Statistics (e.g., "1000+ photos from 50+ events")
- **Design Notes:**
  - Concise and welcoming
  - Optional: Icon or visual element

---

### 3. Search & Filter Options
- **Status:** Required
- **Format:** Search bar + filter options (sticky/top-aligned)
- **Search:**
  - Free-text search box with placeholder: "Search gallery..."
  - Search by photo title, description, or keywords
  - Real-time results as user types
- **Filter Options:**
  - **By Type/Category:** Program, Training, Event, Field Visit, Community, Nature/Trees, Celebration
  - **By Year:** 2024, 2023, 2022, etc. (dropdown with years available)
  - **By Event/Program:** Filter by specific event or program name
  - **By Audience:** Students, Farmers, Women, General Public
  - **By Season:** Spring, Summer, Monsoon, Winter (if applicable)
- **Design Notes:**
  - Search box prominent
  - Filter pills/buttons (toggle on/off)
  - "Reset Filters" button
  - Results count: "Showing X photos"
  - Mobile: Collapsible filter menu or filter drawer

---

### 4. Masonry Grid Layout
- **Status:** Required
- **Format:** Responsive masonry/grid of photos
- **Grid Specifications:**
  - **Desktop (1024px+):** 4-column masonry grid
  - **Tablet (768-1023px):** 3-column grid
  - **Mobile (<768px):** 2-column grid or 1-column with full-width
  - **Gap:** 12-16px between images
  - **Image aspect ratio:** Mixed (not uniform) for masonry effect

**Each Photo Tile:**
- **Image:** Optimized photo (responsive sizing)
- **Hover overlay:** Semi-transparent dark overlay (opacity: 0.6)
- **On hover, show:**
  - Photo title or description text
  - View icon (magnifying glass or expand icon)
  - Optional: Category tag/badge
- **Click action:** Opens photo lightbox/modal with full image
- **Focus state:** Clear outline for keyboard navigation

---

### 5. Lightbox/Modal View
- **Status:** Required
- **Triggered by:** Click on photo in masonry grid
- **Format:** Full-screen lightbox modal
- **Content & Features:**

**Image Display:**
- Full-size photo (optimized for viewing)
- Maintains aspect ratio
- Keyboard navigation: Arrow keys to next/previous
- Touch swipe navigation (mobile devices)

**Photo Information Panel:**
- **Photo title** (H3)
- **Date taken** (e.g., "December 2024" or specific date)
- **Category/type** (Program, Training, Event, etc.)
- **Location** (where photo was taken, if relevant)
- **Description** (1-3 sentences about the photo)
- **Related event/program** (linked if applicable)
- **Photographer credit** (optional, if applicable)

**Navigation Controls:**
- **Previous button:** Arrow left or "← Previous"
- **Next button:** Arrow right or "Next →"
- **Photo counter:** "15 of 45 photos" showing position in gallery
- **Close button:** (X) button to close lightbox
- **Keyboard shortcuts:** Left/Right arrows, Escape to close

**Sharing Options:**
- Social media share buttons (Facebook, Instagram, WhatsApp, Pinterest)
- Copy link button
- Download button (if high-res version available)

**Thumbnail Strip (Optional):**
- Small thumbnail previews of nearby photos at bottom
- Allows quick navigation through related images

## Interactions & Features

### Gallery Grid Interactions
- **Hover:** Semi-transparent overlay appears with title/description
- **Click:** Opens lightbox with full-size image
- **Focus:** Clear focus outline for keyboard navigation

### Search & Filter Interactions
- **Search input:** Real-time filtering as user types
- **Filter pills:** Toggle on/off to apply/remove filters
- **Multiple filters:** AND logic (show photos matching ALL filters)
- **Reset:** Clear all filters, show full gallery
- **Results update:** Grid updates to show matching photos
- **Mobile:** Hamburger menu for filter options

### Lightbox/Modal Interactions
- **Open:** Smooth fade-in animation
- **Navigation:** Arrow keys (left/right), or click arrow buttons
- **Touch:** Swipe left/right for next/previous
- **Close:** Click (X), press Escape, or click outside image
- **Sharing:** Social media sharing without closing modal
- **Download:** Opens download dialog if available

### Responsive Behavior
- **Desktop:** 4-column masonry, full lightbox
- **Tablet:** 3-column grid, touch-optimized lightbox
- **Mobile:** 2-column grid, full-screen lightbox with larger controls

## Design & Visual Approach

### Color Usage
- **Headers:** Sage green (#6B8E23)
- **Search bar:** White background, sage green focus outline
- **Filter pills:** Light gray default, sage green when active
- **Lightbox overlay:** Dark (rgba(0,0,0,0.7)) for contrast
- **Text in lightbox:** White for readability
- **Navigation arrows:** White or sage green (high contrast)

### Layout
- **Max content width:** 1200px (for metadata sections)
- **Masonry grid:** Full viewport width with side padding
- **Grid gap:** 16px (desktop), 12px (mobile)
- **Image sizing:** Responsive, maintain aspect ratio
- **Lightbox:** Full-screen, centered image

### Typography
- **Section heading (H2):** Serif, 36px, sage green
- **Photo title (H3 in lightbox):** Sans-serif, 24px, white
- **Photo metadata:** Sans-serif, 14px, light gray
- **Lightbox text:** White for visibility on dark background

---

## Technical Specifications

### Image Optimization
- **Format:** WebP (primary), JPG (fallback)
- **Sizes:** Create multiple sizes for responsive display
  - Thumbnail: 300x300px
  - Grid view: 600x600px
  - Lightbox: 1200x1200px or full resolution
- **Compression:** Optimized for web loading (< 300KB per image)
- **Lazy loading:** Load images as user scrolls

### Lightbox Library
- **Functionality:** Previous/Next navigation, fullscreen, keyboard controls
- **Options:** Consider Lightbox2, GLightbox, or similar
- **Accessibility:** Keyboard navigation, ARIA labels
- **Touch support:** Swipe gestures for mobile

## Responsive Design

### Desktop (1024px+)
- 4-column masonry grid
- Full-size lightbox with side navigation
- Search and filters side-by-side
- All metadata visible

### Tablet (768px - 1023px)
- 3-column grid
- Touch-friendly lightbox
- Filter options in dropdown or tabs
- Adjusted spacing

### Mobile (< 768px)
- 2-column grid or single-column full-width
- Full-screen lightbox optimized for mobile
- Larger touch targets for controls (44x44px minimum)
- Hamburger menu for filters
- Swipe navigation in lightbox

---

## Accessibility

- **Alt text** for every photo (descriptive)
- **Image titles:** Meaningful, descriptive titles in lightbox
- **Keyboard navigation:** Full gallery and lightbox navigation via keyboard
- **Focus management:** Clear focus indicators
- **Lightbox controls:** Large, easily accessible buttons (44x44px minimum)
- **Color contrast:** White text on dark backgrounds (WCAG AAA)
- **ARIA labels:** For buttons, search box, filter options
- **Skip links:** "Skip to gallery" link for keyboard users
